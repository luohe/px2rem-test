# px2rem-examples

Based on webpack v4.

## How to start

``` bash
npm install
npm run build
open index.html
```

## Things should note

* `webpack.config.js`: the way to config px2rem-loader and other webpack config.
* `foo.css`: the way to use REM unit in CSS source file.
* `index.html`: the way to include `lib.flexible`.
